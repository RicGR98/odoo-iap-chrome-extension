How does it work?

For each key of your choice, you can define multiple options.

Each option has its short name (that will be displayed in the popup as a button) and the actual value that is going to be used.

The special option "show" is a boolean to determine whether to show the option or not (without deleting it entirely)

Don't forget to click on Save to apply your changes and see how the popup behaves in an Odoo instance
